# рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ, ­ЪЉІ, рЃЏрЃћ рЃњрЃљрЃ«рЃџрЃљрЃЋрЃљрЃарЃЌ рЃарЃљрЃЏрЃљрЃќрЃў рЃЏрЃљрЃўрЃАрЃљрЃерЃЋрЃўрЃџрЃў, рЃЋрЃћрЃЉ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃў, рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃБрЃџрЃў рЃърЃћрЃЊрЃљрЃњрЃЮрЃњрЃў



 ## рЃЏрЃћ рЃЋрЃћрЃЉ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃў (Full stack developer), рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃбрЃЮрЃарЃў рЃЊрЃљ рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃў
 - ­ЪЉеРђЇ­ЪЈФ рЃЏрЃћ рЃЏрЃЮрЃЏрЃгрЃЮрЃюрЃА рЃЋрЃћрЃЉ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃБрЃюрЃљрЃарЃћрЃЉрЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃћрЃЉрЃљ, рЃљрЃАрЃћрЃЋрЃћ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЏрЃљрЃЏрЃБрЃџрЃћ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃАрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ (рЃерЃћрЃФрЃџрЃћрЃЉрЃўрЃАрЃЊрЃљрЃњрЃЋрЃљрЃарЃљрЃЊ)
- ­Ъї▒ рЃљрЃЏрЃЪрЃљрЃЏрЃљрЃЊ рЃЋрЃфрЃЊрЃўрЃџрЃЮрЃЉ рЃерЃЋрЃЦрЃЏрЃюрЃљ рЃЮрЃюрЃџрЃўрЃю рЃЎрЃБрЃарЃАрЃћрЃЉрЃў рЃЋрЃћрЃЉ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ 
- ­ЪЉеРђЇ­Ъњ╗ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃћрЃЌ рЃЕрЃћрЃЏрЃў рЃАрЃбрЃљрЃбрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃў
